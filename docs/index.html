Что такое РОСБИОТЕХ?

РОСБИОТЕХ — государственный университет, один из крупнейших в стране. Мы меняем мир через биотехнологии и здоровьесбережение, создавая инновации для успешного будущего.
Это уникальное пространство, где фундаментальные знания, инновационные технологии и современные подходы к образованию открывают широкие горизонты для реализации твоего потенциала. Мы формируем профессионалов нового поколения, способных эффективно решать актуальные задачи настоящего и будущего.
